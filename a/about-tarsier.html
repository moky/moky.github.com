关于“塔希娅”
其实很简单。 就是看中了 github 提供的个性域名服务， 于是想做一个最简单的个人博客， 只需要写几行 HTML + JS 就 OK 的。 想起之前在 Panglao Island 玩的时候曾见过一种很可爱的小动物：眼镜猴， 它只有巴掌大，非常的可爱， 我去看它的时候，它们一个个都抱着一个树枝在那儿睡大觉， 就算你用手去轻抚它，它最多也就懒洋洋地半睁一下眼瞟你一眼， 然后继续睡觉……真是个懒鬼！ 而我要写的这个项目，恰恰也是一个非常小巧，而且简单易用的， 特别适合像我这样的“懒人”， 所以我就用了 Tarsier 来做这个项目的名字。 你看它们，是不是很可爱呀？ 本来想用 SpriteForest 的思想来做的，这样能提供更好的扩展性， 不过想了想，还是觉得先怎么简单怎么来吧， 于是下了两个 jQuery 插件，写了这么个最简单的版本。 将就着看吧～ 呵呵！ 回头我再慢慢更新 *^_^* //---- Updated by moKy @ Dec. 19, 2013 { 一个码农的博客，又怎么能够缺少对代码自动高亮处理的功能呢？ 所以今天特意加了一个 code.js 模块。 作为 demo，先把一个小程序放上来看看效果。 :D //---- } //---- Updated by moKy @ Dec. 23, 2013 { 第一天，上帝说：要有光！ 于是就有了光。 第二天，上帝说：要有天空！ 于是就有了天空。。 第三天，上帝说：要有陆地和海洋！ 于是就有了陆地和海洋。。。 第四天，上帝说：要有太阳和月亮！ 于是就有了太阳和月亮。。。。 第五天，上帝说：要有鱼和飞鸟！ 于是就有了鱼和飞鸟。。。。。 第六天，上帝说：要有动物和人！ 于是就有了动物和人。。。。。。 。。。。。。。 第12天，上帝说：要有评论功能！ 于是。。。。。。。。 //---- }
